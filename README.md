# advent-of-code-2022

## install

    npm install
    npm i -g ts-node 

## run

    # run the latest day, and part, with data from data.txt 
    ts-node index.ts
    
    # run the latest day, and part, with sample data from sample.txt
    ts-node index.ts sample
    
    # other iterations 
    ts-node index.ts day01 part1 
    ts-node index.ts day01 sample
